## Phase 2: 확장 기능 (30개 기능, 6개월)

### M7: 연결 + 네비게이션 (Week 27~34) — 12개

|---|------|------|--------|

| 32 | 호버 미리보기 | Ctrl+호버 시 대상 문서 팝업 (최대 20줄) | SQLite 인덱스 |
| 33 | 링크 자동 갱신 | 파일명 변경 시 모든 참조 파일의 링크 자동 업데이트 | SQLite 인덱스 |

| # | 기능 | 상세 | 의존성 |
|---|------|------|--------|
| 35 | Quick Switcher | Cmd+O, 파일명/별칭/헤딩(`#`)/블록(`^`) 퍼지 검색 | 독립 |
| 36 | 북마크 시스템 | 파일/헤딩/블록 북마크, 사이드바 패널 | 독립 |
| 37 | 뒤로/앞으로 | 브라우저 스타일 점프 히스토리 네비게이션 | 독립 |

| 39 | 문서 전환 | Ctrl+Tab 빠른 문서 전환 | 독립 |

**완료 기준**: `[[문서명]]` 입력 시 자동완성이 동작하고, 백링크가 표시되며, 파일명 변경 시 모든 참조가 자동 갱신된다.
