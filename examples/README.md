# Baram Examples

A small demo vault that showcases Baram's core features. **Open this `examples/`
folder in Baram** (File → Open Folder) to explore it interactively.

`Dijkstra's Algorithm.md` is the main showcase note — it uses headings,
callouts, inline and block math (KaTeX), a table, a Mermaid diagram, a
syntax-highlighted code block, a task list, footnotes, tags, and wikilinks to
the other notes. All four notes are interlinked, so the graph view, backlinks,
and wikilink autocomplete populate automatically.
