## M7 의존성 구조

M7의 **링크 인덱스 (SQLite)** 가 핵심 의존성입니다:

```
SQLite 링크 인덱스 (§7.4)
  ├── 양방향 링크 (#28)
  ├── 백링크 패널 (#29)
  ├── 블록 참조 (#30)
  ├── 링크 자동완성 (#31)
  ├── 호버 미리보기 (#32)
  ├── 링크 자동 갱신 (#33)
  └── 언링크드 멘션 (#34)

독립 작업 (인덱스 불필요):
  ├── Quick Switcher (#35)
  ├── 북마크 (#36)
  ├── 뒤로/앞으로 (#37)
  ├── 멀티 탭 강화 (#38)
  └── 문서 전환 (#39)
```

## 추천 착수 순서

1. **Rust: SQLite 링크 인덱서** — M7 전체의 기반이 되는 핵심 인프라
2. **Wikilink Extension + 파이프라인 변환기** — `[[]]` 파싱/렌더링
3. **백링크 패널 UI** — 인덱서 활용 첫 기능
4. **Quick Switcher / 네비게이션** — 인덱스와 독립적이므로 병렬 가능
