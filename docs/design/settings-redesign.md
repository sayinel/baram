# Settings 고도화 설계서

> **상태**: 승인됨
> **작성일**: 2026-03-08
> **우선순위**: P0 → P1 → P2 순차 구현

---

## 1. 현재 상태 진단

### 문제점 요약

| 영역 | 현재 상태 | 문제 |
|------|-----------|------|
| **탭 구조** | 8개 탭 (General/Editor/Appearance/Files/Markdown/Extensions/Workspace/AI) | 카테고리 경계가 모호 (Files vs Markdown, General에 Journal 설정이 혼재) |
| **Extensions 탭** | codeBlock, mermaidBlock 2개 설정만 존재 | Markdown extension 설정인데 독립 탭으로 있을 이유 부족 |
| **Theme 미리보기** | 카드에 5개 색상 스와치만 표시 | 실제 에디터 모습 예측 불가 |
| **Activity Bar** | 상단 11개 + 하단 6개 = 17개 아이콘 | 점점 복잡, 커스터마이즈 불가 |
| **플러그인** | 미존재 | 설치/삭제/설정 관리 UI 없음 |
| **i18n** | 없음, 모든 문자열 하드코딩 | 다국어 지원 불가 |

## 2. 새 카테고리 구조

### 현재 → 제안 비교

```
현재 (8탭 모달)                    제안 (사이드 네비 + 검색)
─────────────────                 ──────────────────────────
General ─────────┐                🔍 검색
Editor           │                ──────────
Appearance       │ 모달 탭 전환     일반 (General+Files)
Files            │                에디터
Markdown         │                외관 (Appearance+Workspace)
Extensions       │                마크다운 (Markdown+Extensions)
Workspace        │                AI
AI ──────────────┘                ──────────
                                  플러그인 (P2, 신규)
                                  Activity Bar (P1, 신규)
                                  단축키 (P2, 신규)
                                  언어 (P1, 신규)
```

### 카테고리별 설정 배치

| 카테고리 | 포함 설정 | 변경사항 |
|---------|----------|---------|
| **일반** | 시작 동작, 자동 저장, 맞춤법 검사, 링크 형식, 파일 관련 설정, Journal, 스냅샷 | General + Files 병합 |
| **에디터** | 폰트, 줄 높이, 탭 크기, 줄번호, 괄호 자동 완성, 최대 폭, 줌 | 기존과 동일 |
| **외관** | 테마 갤러리 (개선된 미리보기), 워크스페이스 프리셋 | Appearance + Workspace 병합 |
| **마크다운** | 인라인 수식, 하이라이트, 취소선, 코드블록 스타일/줄번호, 다이어그램 등 | Markdown + Extensions 병합 |
| **AI** | 프로바이더, 모델, API 키, Ghost Text, 커스텀 AI 명령어 | 기존과 동일 |
| **플러그인** | 플러그인 탐색/설치/삭제, 플러그인별 설정 | **P2 신규** |
| **Activity Bar** | 아이콘 표시/숨기기, 순서 변경, 위치(상/하) 지정 | **P1 신규** |
| **단축키** | 키보드 단축키 목록 및 커스터마이즈 | **P2 신규** |
| **언어** | 인터페이스 언어 선택 (EN/KR) | **P1 신규** |

## 4. 테마 미리보기 개선

### 미니 에디터 프리뷰

```
┌─────────────────────────────────────────────┐
│  Tokyo Night                          [적용] │
│  ┌─────────────────────────────────────────┐ │
│  │ ▫ sidebar │  # Heading              ─┐  │ │
│  │ ▫ file.md │  Some **bold** text     │  │ │
│  │ ▫ note.md │  > A blockquote         │  │ │
│  │           │  ```code block```       │  │ │
│  │           │  - list item            ─┘  │ │
│  └─────────────────────────────────────────┘ │
│  🌙 Dark  ·  #7aa2f7 accent                │
└─────────────────────────────────────────────┘
```

- 각 테마 카드에 미니 에디터 모형 렌더링 (사이드바/에디터/본문 스타일 포함)
- 실제 CSS 변수를 적용한 라이브 프리뷰
- 카드 크기를 키워서 한 줄에 2~3개 배치

## 6. Activity Bar 커스터마이즈 (P1)

- 체크박스: 아이콘 표시/숨기기
- 드래그 핸들: 순서 변경
- 설정 아이콘은 항상 표시 (비활성화 불가)
- 플러그인이 Activity Bar 아이콘 등록 가능

---

## 7. 언어 설정 (P1)

- `src/i18n/` 디렉토리에 `en.json`, `ko.json` 번역 파일
- 경량 i18n 라이브러리 또는 자체 구현
- `useTranslation()` 훅으로 컴포넌트에서 사용
- 번역 키 네임스페이스: `settings.*`, `sidebar.*`, `editor.*`, `common.*`
- 커뮤니티 번역 파일을 추가 가능한 개방적 구조

---

## 8. 설정 검색

- 설정 라벨, 설명, 카테고리 경로를 모두 검색 대상에 포함
- 검색 결과는 해당 설정 컨트롤을 바로 인라인으로 표시 (VS Code Settings 방식)
- 빈 검색 시 기존 카테고리 뷰로 복귀

---

## 9. 구현 우선순위

| 우선순위 | 항목 | 이유 |
|---------|------|------|
| **P0** | 카테고리 재구성 + 사이드 네비 전환 + 설정 검색 | 기존 설정의 접근성 즉시 개선 |
| **P0** | 테마 미니 프리뷰 | 외관 설정의 핵심 UX 문제 |
| **P1** | Activity Bar 커스터마이즈 | 아이콘 증가 문제에 대한 해결책 |
| **P1** | i18n 기반 + 언어 설정 | 구조적 변경이므로 초기에 넣어야 나중에 비용 절감 |
| **P2** | 플러그인 관리 UI | 플러그인 시스템 자체 개발이 선행되어야 함 |
| **P2** | 단축키 커스터마이즈 | nice-to-have |
